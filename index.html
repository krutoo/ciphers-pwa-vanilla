Шифр Цезаря

Мультипликативный шифр


Подсказка

В мультипликативном шифре алгоритм шифрования применяет умножение исходного текста ключом, а алгоритм дешифрования применяет деление зашифрованного текста ключом.


Ограничения

  • Символы, не встречающиеся в алфавите, не шифруются;
  • значение ключа должно быть выбрано таким, что ключ и длина алфавита - взаимно простые числа;
  • модуль ключа шифра не должен превышать длину алфавита.

Афинный шифр


Подсказка

Формула для шифрования символа:
y = (ax + b) % n, где

  • a, b - компоненты шифра;
  • n - длина алфавита;
  • x - исходная позиция символа в алфавите.

Формула для расшифровки символа:
x = (a-1 * (y - b)) % n, где

  • a-1 - обратное по модулю n число к a;
  • y - позиция зашифрованного символа в алфавите.

Ограничения

  • Символы, не встречающиеся в алфавите, не шифруются;
  • значение a должно быть выбрано таким, что a и n - взаимно простые числа;
  • модуль компонента ключа шифра b не должен превышать длину алфавита.
